An execution step of the EVM.

§exec_state: ExecState

Execution state

§pc: u64

Program Counter

§stack_size: usize

Stack size

§memory_size: usize

Memory size

§gas_left: u64

Gas left

§gas_cost: u64

Gas cost of the step. If the error is OutOfGas caused by a “gas uint64 overflow”, this value will not be the actual Gas cost of the step.

§gas_refund: u64

Accumulated gas refund

§call_index: usize

Call index within the Transaction.

§rwc: RWCounter

The global counter when this step was executed.

§reversible_write_counter: usize

Reversible Write Counter. Counter of write operations in the call that will need to be undone in case of a revert. Value at the beginning of the step.

§reversible_write_counter_delta: usize

Number of reversible write operations done by this step.

§log_id: usize

Log index when this step was executed.

§bus_mapping_instance: Vec<OperationRef, Global>

The list of references to Operations in the container

§copy_rw_counter_delta: u64

Number of rw operations performed via a copy event in this step.

§error: Option<ExecError>

Error generated by this step
